## Installation Instructions

### Windows
1. Download the `.exe` installer
2. Run the installer and follow the prompts
3. Launch Ter-Plus-Plus from your Start menu

### macOS
1. Download the `.dmg` file
2. Open the `.dmg` and drag Ter-Plus-Plus to Applications
3. Launch from Applications folder

### Linux (AppImage)
1. Download the `.AppImage` file
2. Make it executable: `chmod +x Ter-Plus-Plus-*.AppImage`
3. Run: `./Ter-Plus-Plus-*.AppImage`

### Linux (Debian/Ubuntu)
```bash
sudo dpkg -i ter-plus-plus_*.deb
```

### Linux (Fedora/RHEL)
```bash
sudo dnf install ter-plus-plus-*.x86_64.rpm
```
